# Odin Landing Page

This project is a practice landing page created as part of **The Odin Project** curriculum.

The goal of this assignment was to practice building a basic landing page layout using **HTML and CSS**, focusing on structure, layout, and styling rather than functionality.

## Overview

- Built with plain HTML and CSS
- Based on the provided Odin Project design template
- Customized colors, font styles, and font sizes
- No JavaScript included

Placeholder images were sourced from [picsum.photos](https://picsum.photos/).

## Purpose

This project serves as a learning exercise to reinforce:

- Flexbox layout
- Page structure and spacing
- Consistent styling and typography
- Translating a design reference into code
